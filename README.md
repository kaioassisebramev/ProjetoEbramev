# Projeto4

Sistema desenvolvido com Next.js 14, NextAuth v4, Prisma e TypeScript.

## Stack Tecnológica

- **Framework:** Next.js 14 (App Router)
- **Autenticação:** NextAuth v4.24.7
- **ORM:** Prisma 5.19.1
- **Banco de Dados:** SQLite
- **Linguagem:** TypeScript

## Instalação

1. Instale as dependências:
```bash
npm install
```

2. Configure o banco de dados:
```bash
npm run db:push
```

3. Popule o banco com dados iniciais:
```bash
npm run db:seed
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```
